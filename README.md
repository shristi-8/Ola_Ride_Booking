# 🚖 Ola Ride Booking Analysis

## 📌 Project Overview

This project analyzes Ola ride booking data to uncover booking trends, customer behavior, revenue performance, ride cancellations, and operational efficiency. The analysis was performed using Python for data cleaning and EDA, MySQL for business queries, and Power BI for interactive dashboard creation.

---

## 🎯 Project Objective

- Analyze ride booking patterns and customer behavior.
- Identify revenue trends and operational performance.
- Understand ride cancellations and their reasons.
- Build an interactive dashboard for business decision-making.
- Generate actionable business insights using data analytics.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- MySQL
- Power BI
- VS Code
- Git & GitHub

---

## 📂 Dataset

The dataset contains **150,000+ ride booking records** with information such as:

- Booking Status
- Booking Value
- Ride Distance
- Vehicle Type
- Pickup & Drop Locations
- Customer & Driver Ratings
- Payment Method
- Cancellation Reasons
- Booking Date & Time

---

## 🧹 Data Cleaning

The following preprocessing steps were performed:

- Removed duplicate records
- Converted data types
- Checked missing values
- Cleaned text columns
- Created Month, Day, Hour, and Ride Status columns
- Prepared data for SQL analysis and Power BI dashboard

---

## 📊 Exploratory Data Analysis (EDA)

The project answers business questions such as:

- Total bookings by booking status
- Revenue by vehicle type
- Peak booking hours
- Ride demand by day of week
- Payment method distribution
- Ride distance analysis
- Customer and driver rating analysis
- Cancellation analysis
- Revenue trends
- Booking trends over time

---

## 🗄️ SQL Analysis

Performed **30 business-focused SQL queries** using MySQL, including:

- Aggregate Functions
- GROUP BY & HAVING
- CASE WHEN
- Window Functions
- CTEs
- Ranking Functions
- Business KPI Analysis

---

## 📈 Power BI Dashboard

Created an interactive dashboard featuring:

- KPI Cards
- Revenue Analysis
- Booking Trends
- Cancellation Analysis
- Customer & Driver Ratings
- Vehicle Type Analysis
- Peak Booking Hours
- Interactive Filters & Slicers

## 💡 Key Business Insights

- Identified peak booking hours and demand patterns.
- Analyzed vehicle-wise revenue contribution.
- Discovered major customer and driver cancellation reasons.
- Evaluated customer satisfaction using ratings.
- Identified high-demand pickup and drop locations.
- Measured overall business performance using KPIs.
